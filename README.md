# SQL Sales Analysis Project

A beginner-level SQL project analyzing sales data using MySQL.

## Database Structure

- **customers** — customer_id, customer_name, city, country
- **products** — product_id, product_name, category, price
- **orders** — order_id, customer_id, product_id, quantity, order_date

## Analysis Performed

- Total Revenue Calculation
- Revenue by Product Category
- Top 5 Products by Revenue
- Top 5 Customers by Spending
- Monthly Revenue Trend (Jan–Jun 2024)

## SQL Concepts Used

- JOIN (INNER JOIN across multiple tables)
- GROUP BY & ORDER BY
- Aggregate Functions (SUM, COUNT)
- DATE_FORMAT for monthly trends
- FOREIGN KEY constraints
